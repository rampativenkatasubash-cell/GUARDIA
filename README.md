# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/7a79aae3-1b71-4390-be04-5fd815246479

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `PROTECTACT_API_KEY` in [.env.local](.env.local) to your protectAct API key
3. Run the app:
   `npm run dev`
